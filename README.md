# 🚗 Vehicle Online Sales Trends Analysis
# 📌 Project Overview

This project analyzes vehicle registration data to uncover trends in vehicle types, fuel usage, manufacturer performance, and customer preferences.
The goal is to transform raw data into meaningful insights that support data-driven decision-making in the automotive domain.

# 🎯 Problem Statement

Large volumes of vehicle registration data are generated, but they are often not effectively analyzed to identify patterns and trends.
This project aims to answer:
Which vehicle types are most popular?
What fuel types dominate the market?
Which manufacturers lead in registrations?
How can we derive business insights from this data?

# 🛠️ Tools & Technologies Used
Python (Pandas, NumPy)
Matplotlib & Seaborn (Data Visualization)
Power BI (Dashboard Creation)

# ⚙️ Data Preprocessing
Handled missing values
Removed duplicate records
Converted date columns into proper format
Standardized categorical values
Created new features:
year, month
registration_duration
vehicle_condition (New/Used)

# 📈 Key Insights
Petrol vehicles dominate the market
Two-wheelers have the highest number of registrations
A few manufacturers control a large market share
Transport vehicles contribute higher transport scores
Registration trends vary over time

# 💡 Strategic Insights
Focus on alternative fuel vehicles (EV, CNG)
Target high-demand segments like two-wheelers
Improve strategies to compete with dominant manufacturers
Optimize offerings for high seat-capacity vehicles
Use trends for better inventory and pricing decisions

# 🧾 Conclusion

This project provides a comprehensive analysis of vehicle registration trends by examining vehicle categories, fuel types, and manufacturer performance.
It highlights key market patterns and demonstrates how data analytics can transform raw data into actionable insights.
The derived transport score adds additional business value by evaluating vehicle utility.
Overall, the project showcases the importance of data-driven decision-making in the automotive industry.
