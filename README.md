# 🚗 Vehicle Online Sales Trends Analysis

---

## 📌 Project Overview
This project analyzes vehicle registration data to uncover trends in **vehicle types, fuel usage, manufacturer performance, and customer preferences**.  
The goal is to transform raw data into **meaningful insights** and support **data-driven decision-making** in the automotive domain.

---

## 🎯 Problem Statement
Large volumes of vehicle registration data are generated, but they are not effectively analyzed.  
This project helps to identify:
- Most popular vehicle types  
- Fuel usage trends  
- Top-performing manufacturers  
- Key insights for business decisions  

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy)  
- Matplotlib & Seaborn  
- Power BI  

---

## ⚙️ Data Preprocessing

- Handled missing values  
- Removed duplicates  
- Converted date formats  
- Standardized categorical data  
- Feature Engineering:
  - Year, Month  
  - Registration Duration  
  - Vehicle Condition (New/Used)  

---

## 📊 Exploratory Data Analysis

### 🔹 Univariate Analysis
- Fuel distribution  
- Vehicle class distribution  

### 🔹 Bivariate Analysis
- Vehicle class vs fuel  
- Manufacturer vs registrations  

### 🔹 Multivariate Analysis
- Relationship between seat capacity, value score, and category  

---

## 📈 Key Insights

- Petrol vehicles dominate the market  
- Two-wheelers have the highest registrations  
- Few manufacturers control major market share  
- Transport vehicles have higher transport scores  
- Registration trends vary over time  

---

## 💡 Strategic Recommendations

- Focus on alternative fuel vehicles  
- Target high-demand segments like two-wheelers  
- Improve competition with leading manufacturers  
- Optimize high seat-capacity vehicles  
- Use trends for pricing and inventory decisions

This project provides a comprehensive analysis of vehicle registration trends by examining vehicle categories, fuel types, and manufacturer performance.
